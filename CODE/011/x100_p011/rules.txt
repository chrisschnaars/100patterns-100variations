100X100 Patterns
No. 011: Arcs 2


RULES

1. SHAPE
EACH CELL CAN HAVE THE FOLLOWING SHAPES
 0: BLANK CELL
 1: HALF ARC
 2: ARC + RECTANGLE
 3: FULL ELLIPSE
 4: QUARTER ARC

2. ORIENTATION
 0: FACE NORTH
 1: FACE EAST
 2: FACE WEST
 3: FACE SOUTH

2. COLOR
 EACH SHAPE GETS RANDOMLY SELECTED COLOR FROM THE LIST

3. SCALE
 ALL SHAPES DRAWN TO SAME SCALE
