100X100 Patterns
No. 013: Shapes and Strokes


RULES

1. SHAPE
 EACH CELL HAS ONE SHAPE
  CIRCLE
  RECTANGLE
  TRIANGLE

1B. TRIANGE ORIENTATION
 TRIANGLES CAN BE ORIENTED ALONG ANY AXIS OF THE CELL

3. STROKE WIDTH/FILL
 EACH SHAPE CAN HAVE A LIGHT STROKE, MED STROKE, HEAVY STROKE, OR FILL
 
2. COLOR
 EACH SHAPE GETS RANDOMLY SELECTED COLOR FROM THE LIST
